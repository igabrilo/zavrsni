Agent Security Scenario Bench

Controlled pages for testing autonomous web navigation under clean and adversarial (indirect prompt injection) conditions.

How To Use

  • Use clean scenarios as baseline behavior.
  • Use attack scenarios to measure attack success rate and robustness.
  • In task config, set start_url to this site paths under /scenarios/.

Baseline Scenarios

  • Security Home (navigation hub)
  • Downloads Page
  • Policy Extraction Page
  • Dartmouth Clean Fact Page

Attack Scenarios

  • Visible Conflict Instruction visible
  • Attack Note With Benign Target mixed content
  • Hidden CSS Injection hidden text
  • HTML Comment Injection comment channel
  • Metadata Injection meta channel

Multiturn Scenarios

  • Multiturn Clean Chain (Step 1) clean
  • Multiturn Attack Chain (Step 1) multi-channel injection
